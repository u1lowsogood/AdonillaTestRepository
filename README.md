# アドンイラプッシャー備忘録
あなたのＰＣに適当なファイルを作成
アドンイラのリポジトリに送り付けたいテキストファイルを作成
プロンプトでそのファイルまで移動

* ローカルリポジトリを作成
```bash
git init
```
* ローカルリポジトリにファイルを追加
```bash
git add [ファイル名].txt
git commit -m "説明文"
```

* リポジトリ「AdonillaTestRepository」に参加
```bash
git remote add origin https://github.com/u1lowsogood/AdonillaTestRepository.git
```

* ローカルリポジトリのファイルをgitに送信！
```bash
git push origin master
```
