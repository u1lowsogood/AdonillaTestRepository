#アドンイラプッシャー備忘録

あなたのＰＣに適当なファイルを作成
アドンイラのリポジトリに送り付けたいテキストファイルを作成
プロンプトでそのファイルまで移動

#ローカルレポジトリを作成
git init

#ローカルレポジトリにファイルを追加
git add [ファイル名].txt
git commit -m "説明文"

#リポジトリ「AdonillaTestRepository」に参加？
git remote add origin https://github.com/u1lowsogood/AdonillaTestRepository.git

#ローカルレポジトリのファイルをgitに送信！
git push origin master
