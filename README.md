# アドンイラプッシャー備忘録

* ローカルリポジトリを作成
```bash
git init
```
* ローカルリポジトリにファイルを追加
```bash
git add [ファイル名].txt
git commit -m "説明文"
```

* リモートリポジトリ「AdonillaTestRepository」に参加
```bash
git remote add origin https://github.com/u1lowsogood/AdonillaTestRepository.git
```

* ローカルリポジトリのファイルをリモートリポジトリ「AdonillaTestRepository」に送信！
```bash
git push origin master
```
